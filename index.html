📖 ไวยากรณ์เริ่มจาก 0

    📘 รูปสุภาพ (～ます)

    • たべます = กิน
    • のみます = ดื่ม
    • いきます = ไป

    ❓ か (คำถาม)

    • これはなんですか？ = อันนี้คืออะไร?
    • なんさいですか？ = อายุเท่าไรครับ

    🔹 は (หัวข้อประโยค)

    • わたしはフランです。 = ฉันคือฟราน
    • これはほんです。 = นี่คือหนังสือ

    🔹 を (กรรม)

    • ごはんをたべます。 = กินข้าว
    • みずをのみます。 = ดื่มน้ำ

    🔹 に (เวลา / ปลายทาง)

    • がっこうにいきます。 = ไปโรงเรียน
    • 7じにおきます。 = ตื่นตอน 7 โมง

    🔹 で (สถานที่ที่ทำกิจกรรม)

    • いえでたべます。 = กินที่บ้าน
    • がっこうでべんきょうします。 = เรียนที่โรงเรียน

    🔹 の (ของ / ขยายคำนาม)

    • わたしのほん = หนังสือของฉัน
    • にほんのえいが = หนังญี่ปุ่น

    🔹 も (ก็ / ด้วย)

    • わたしもがくせいです。 = ฉันก็เป็นนักเรียน
    • みずものみます。 = ดื่มน้ำด้วย
